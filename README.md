# Clone Login X (Twitter)

Interface inspirada na página de login e cadastro do X (antigo Twitter), desenvolvida com foco em estudos de front-end e reprodução de interfaces modernas.

---

## 📖 Sobre o Projeto

Este projeto foi criado com o objetivo de aprimorar habilidades em desenvolvimento front-end, praticando conceitos como:

* Estruturação semântica com HTML5
* Estilização moderna com CSS3
* Responsividade
* Organização de layout
* Fidelidade visual de interfaces

---

## ⚠️ Disclaimer

Este repositório possui finalidade exclusivamente educacional.

O projeto não realiza autenticação real, armazenamento de dados ou qualquer tipo de coleta de informações de usuários.

Não há qualquer intenção de phishing, engenharia social ou uso malicioso.
Todas as referências visuais, marcas e identidades pertencem aos seus respectivos proprietários.

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3

---

## 🛠️ Como Executar

Clone o repositório:

```bash
git clone https://github.com/Gabrielsuz/Clone-Login-X-antigo-twitter-.git
```

Acesse a pasta do projeto:

```bash
cd Clone-Login-X-antigo-twitter-
```

Abra o arquivo `index.html` em seu navegador.
